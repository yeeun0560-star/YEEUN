# 박예은 · 서비스 기획 포트폴리오

고객의 눈으로 보고, 고객의 입장에서 생각하는 서비스 기획자 박예은의 포트폴리오 웹사이트입니다.

## 소개

유통(백화점) 도메인에서 진행한 디지털 사업 기획 · 앱 서비스 기획 · UI/UX 기획 프로젝트를 담았습니다. 각 프로젝트를 클릭하면 실제 기획 슬라이드를 좌우 스와이프로 볼 수 있습니다.

## 주요 프로젝트

1. 우수고객 모바일 서비스
2. 타임빌라스 마이크로 사이트
3. APP Push 활성화 마케팅
4. 롯데백화점 앱 리뉴얼
5. 지점별 쇼핑정보 서비스 (Shop Now)
6. 인터랙티브 매장안내 키오스크

## 기술

- HTML · CSS · Vanilla JavaScript (빌드 도구 없이 동작)
- 반응형 레이아웃, 스크롤 애니메이션, 슬라이드 캐러셀 · 라이트박스

## 실행

`index.html` 파일을 브라우저로 열면 됩니다.

## 파일 구조

```
index.html      메인 페이지
styles.css      스타일
script.js       프로젝트 데이터 · 인터랙션
images/         프로젝트 슬라이드 이미지
```
